How to create a xcode template

* create a template directory under ~/Library/Developer/Xcode/Templates/File Templates/Custom:<br>
$ mkdir -p ~/Library/Developer/Xcode/Templates/File Templates/Custom/leetcode.xctemplate

* cp \_\_\_FILENAME\_\_\_.cpp and TemplateInfo.plist under ~/Library/Developer/Xcode/Templates/File Templates/Custom/leetcode.xctemplate
